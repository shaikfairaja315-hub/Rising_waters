# 🌊 Flood Prediction Using Machine Learning

## 📌 Project Overview

This project predicts the possibility of floods using Machine Learning techniques. The goal is to analyze flood-related data, preprocess it, train a predictive model, and generate accurate flood predictions.

## 🎯 Objectives

* Analyze the flood dataset.
* Perform data preprocessing and feature scaling.
* Train and evaluate a Machine Learning model.
* Predict flood risk based on input data.

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* XGBoost
* Jupyter Notebook
* VS Code

## 📂 Project Structure

```text
Flood-Prediction/
│── project.ipynb      # Complete ML workflow
│── main.py            # Main application file
│── dataset.csv        # Dataset
│── README.md          # Project documentation
```

## 🚀 Project Workflow

1. Import required libraries.
2. Load the dataset.
3. Perform data preprocessing.
4. Conduct Exploratory Data Analysis (EDA).
5. Apply feature scaling.
6. Train the Machine Learning model.
7. Evaluate model performance.
8. Predict flood risk using the trained model.

## ✨ Features

* Data preprocessing and cleaning
